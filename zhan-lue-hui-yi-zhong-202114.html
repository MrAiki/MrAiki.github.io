戦略会議中2021(14)

Date Sun 11 July 2021 Tags Lossless Audio Codec / Wavelet / Lifting

Adaptive Polyphase Subband Decomposition Structures for Image Compression を引用している文献を漁る。改善したやつ、音声に適用したやつが出てくるはず。

  • Direction-Adaptive Discrete Wavelet Transform for Image Compression
    • 画像の方位性に着目したウェーブレットの適応。
  • Geometrical grouplets
    • 勾配ベースで適応する手法は、直交していないし、安定性(stability)がしばしば保証されてないという指摘。
    • Direction-Adaptive Discrete Wavelet Transform via Directional Lifting and Bandeletization が源流のようだが、これも得られるウェーブレットが直交してないと指摘。
  • A 2-D Orientation-Adaptive Prediction Filter in Lifting Structures for Image Coding
    • (5,3)を斜め方向（±45°）に対応したフィルタを作った。画像に特化しすぎ？
  • Design of signal-adapted multidimensional lifting scheme for lossy coding
    • 2乗誤差最小化+直交制約をつけてラグランジュ未定乗数で解く。
    • ロッシー向けと言っているが有益そう。
    • まだちゃんと読めてない。。
  • Building nonredundant adaptive wavelets by update lifting
    • 適応リフティングを通じてウェーブレットを設計する手段を提案？
  • Vector-Lifting Schemes for Lossless Coding and Progressive Archival of Multispectral Images
  • Image denoising algorithm via best wavelet packet base using Wiener cost function
    • Wienerコスト、つまり誤差分散最小化。
  • ADAPTIVE DISCRETE GENERALIZED LIFTING FOR LOSSLESS COMPRESSION
    • これも決定写像ベースに見える。
  • ADAPTIVE MULTIRESOLUTION DECOMPOSITION: APPLICATION TO LOSSLESS IMAGE COMPRESSION
    • 適応的フィルタベースのロスレス静止画圧縮手法。RLSで適応。
    • ICASSPなので短い。気になる。
  • LMS BASED ADAPTIVE PREDICTION FOR SCALABLE VIDEO CODING
    • 動画の動き補償の文脈で適応フィルタを使用している？
  • Efficient Least Absolute Deviation Adaptive Wavelet Filter Bank
    • 誤差絶対値基準で最小化。
  • Lossless image compression by LMS adaptive filter banks
    • 短いけど、まさに自分がイメージしていること。適応フィルタ学習によるロスレス圧縮。

音声への適用が見つからないのはどういうこと？確かに2013ICASSPでもウェーブレットパケットで満足してたから少ないのはうなずけるけど不穏かも。
